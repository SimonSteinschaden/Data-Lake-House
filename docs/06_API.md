# API

Aktuell enthält dieses Repository keine implementierte API.
Es existieren im Quellcode keine ASP.NET Core Controller, keine Minimal APIs und keine HTTP-Endpunkte.

Geplante Architekturdetails beschreiben eine zukünftige ASP.NET Core Web API mit Swagger/OpenAPI.
Derzeit sind im Code nur die Domain-, Application- und Infrastructure-Schichten vorhanden.
